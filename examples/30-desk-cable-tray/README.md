This folder contains a **working example** that generates a STEP model for: A cable tray with pattern-generated ribs and mounting options.

The intent is that you can run the code here to emit a STEP file, open it in a CAD viewer, and/or import it into your slicer to 3D print and iterate.

## What this example demonstrates
- arraying features from data
- layout rules (spacing, margins)
- family generation from a single config
- mount geometry derived from standards

## Parameters to try
- `rows`
- `cols`
- `cellSize`
- `holeSpacing`
- `mountThickness`

## Suggested extensions
- generate multiple sizes in one run
- add embossed labels for each pocket
- add cable routing features
